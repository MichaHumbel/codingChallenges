Challenge: return a new sorted merged list from K sorted lists, each with size N.

For example, if we had [[10, 15, 30], [12, 15, 20], [17, 20, 32]], the result should be [10, 12, 15, 15, 17, 20, 20, 30, 32].

Level: EASY

To run a challenge type: node Challenges/CHALLANGEFOLDER NAME 
Example to run Challange in Folder C1: node Challenges/C1
To run a solution: node Challenges/FOLDER/solution
